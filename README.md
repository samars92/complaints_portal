# complaints_portal

Complaints portal for both customers and admin

how to run:
1- git clone https://github.com/samars92/complaints_portal.git
2- cd complaints_app

Backend ->
1- cd backend
2- pip install requirements.txt
3- flask db stamp head 
4- flask db migrate
5- flask db upgrade
6- flask run

Fronted ->
1- cd frontend
2- npm install
3- npm run
